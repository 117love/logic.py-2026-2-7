# よりそいホットラインと自殺予防いのちの電話を足したようなアプリ

よりそいホットラインや自殺予防いのちの電話の様に、人の悩みやストレスに心から寄り添う事を目的としたアプリです。

ユーザーが日常生活で癪に触った出来事や嫌な出来事を入力すると、出来事を整理し励ましや前向きになれたり勇気づけられる様なメッセージを返します！

## 主な機能

・出来事の入力
（いつ・どこで・誰から・どんな言動・どんな気分）

励ましのメッセージの作成

話し方選択
（敬吾・タメ語・混合（敬語とタメ語の）・関西弁）

禁止語句設定
（言われたら嫌な言葉が返答されない機能）
